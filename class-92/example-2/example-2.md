# Beginner Event-Driven Programming Examples

This document contains beginner-friendly examples to help students practice
**Event-Driven Programming** using **HTML, CSS, and JavaScript**.

Each exercise introduces a different type of user interaction and reinforces
important concepts such as:

- Event listeners
- DOM manipulation
- User interaction
- Dynamic updates
- JavaScript events

---

# Example 2: Background Color Changer

## Problem Description

A company wants a simple webpage where users can click a button to change the
background color of the page.

When the user presses the button:

1. The background color should change.
2. A message should appear indicating that the color changed.

This example teaches students how to:

- Listen for click events
- Modify CSS dynamically with JavaScript
- Update text content
