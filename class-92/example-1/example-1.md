# Beginner Example: Button Click Counter

## Problem Description

You have been hired to develop a very simple interactive webpage for a small technology training center.

The center wants a webpage with a button that allows users to count how many times they have clicked on it.

Every time the user presses the button:

1. The click counter should increase by **1**.
2. The webpage should display the updated number of clicks.
3. A message should appear informing the user that the button was clicked.

This example introduces students to the fundamental concepts of **event-driven programming** using:

- HTML for structure
- CSS for styling
- JavaScript for behavior
- Events and event listeners

Students will learn:

- How to select HTML elements using JavaScript
- How to listen for events using `addEventListener()`
- How to respond to user interactions
- How to update content dynamically
- How event-driven programming works in practice

---

# Event-Driven Programming Concepts Used

This beginner example introduces the following concepts:

| Concept           | Example              |
| ----------------- | -------------------- |
| Event             | `click`              |
| Event Listener    | `addEventListener()` |
| Callback Function | `function () {}`     |
| DOM Manipulation  | `textContent`        |
| Variables         | `counter`            |
| User Interaction  | Button click         |

---

# Expected Result

When students run the project:

1. The page displays a button.
2. The counter starts at `0`.
3. Each button click increases the counter.
4. The message changes after interaction.
5. Everything updates instantly without refreshing the page.
